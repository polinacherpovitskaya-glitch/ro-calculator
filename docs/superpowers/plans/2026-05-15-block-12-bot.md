## Task 2: Заменить Supabase-клиент на pg

В `ops/bot/` найти все места использования Supabase (`createClient` из `@supabase/supabase-js`, или прямые HTTP-запросы к Supabase REST). Заменить на наш Postgres через `pg`-driver или на наш `/api/` (через fetch).

Подход: использовать наш собственный API `/api/tasks`, `/api/auth`, и т.п. Тогда бот не имеет прямого доступа к БД — это секурнее.

Но: бот часто работает по `cron`-расписанию и может не иметь cookie-сессии. Решение:
- Добавить в БД таблицу `bot_tokens (token TEXT PRIMARY KEY, name TEXT, created_at TIMESTAMPTZ)`.
- В `ops/api/src/middleware/auth.js` поддержать заголовок `Authorization: Bearer <token>` параллельно с cookie.
- Бот шлёт `Authorization: Bearer <token>` где token прописан в `.env`.

Migration:

```sql
-- 010_bot_tokens.sql
CREATE TABLE IF NOT EXISTS bot_tokens (
    token         TEXT PRIMARY KEY,
    name          TEXT NOT NULL,
    role          TEXT NOT NULL DEFAULT 'bot' CHECK (role IN ('bot','admin')),
    created_at    TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    last_used_at  TIMESTAMPTZ
);

INSERT INTO app_meta (id, version) VALUES (1, '010-bot-tokens')
ON CONFLICT (id) DO UPDATE SET version = EXCLUDED.version, applied_at = NOW();
```

В middleware auth.js:

```js
export async function requireAuth(req, res, next) {
  // Try Bearer first
  const authHeader = req.get('Authorization') || '';
  if (authHeader.startsWith('Bearer ')) {
    const token = authHeader.slice(7).trim();
    const { rows } = await getPool().query(
      `SELECT name, role FROM bot_tokens WHERE token = $1`, [token]
    );
    if (rows[0]) {
      req.user = { id: 0, email: `bot:${rows[0].name}`, role: rows[0].role, employeeId: null, mustChangePassword: false };
      await getPool().query(`UPDATE bot_tokens SET last_used_at = NOW() WHERE token = $1`, [token]);
      return next();
    }
  }
  // Fallback to cookie session (existing logic)
  // ...
}
```

## Task 3: Dockerfile + docker-compose service

```dockerfile
# ops/bot/Dockerfile
FROM node:20-alpine AS deps
WORKDIR /app
COPY package.json package-lock.json ./
RUN npm ci --omit=dev

FROM node:20-alpine AS run
WORKDIR /app
ENV NODE_ENV=production
COPY --from=deps /app/node_modules ./node_modules
COPY package.json ./
COPY . .
VOLUME ["/app/state"]
CMD ["node", "taskbot.js"]
```

В `ops/infra/docker-compose.yml` добавить:

```yaml
  bot:
    build:
      context: ../bot
      dockerfile: Dockerfile
    image: ops-bot:local
    container_name: ops-bot
    restart: unless-stopped
    environment:
      TG_BOT_TOKEN: ${TG_BOT_TOKEN}
      OPS_API_URL: http://api:3000
      OPS_BOT_TOKEN: ${OPS_BOT_TOKEN}
    volumes:
      - bot-state:/app/state
    depends_on:
      - api
    networks:
      - internal

volumes:
  bot-state:
```

В `.env.example` добавить:
```
TG_BOT_TOKEN=<token from BotFather>
OPS_BOT_TOKEN=<сгенерировать openssl rand -hex 32>
```
