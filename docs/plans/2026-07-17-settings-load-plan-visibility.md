# План: видимый план загрузки в настройках

## 1. Эффективный сезонный план

- [ ] Добавить источник сезонных процентов: явный план или вычисленное
  равномерное распределение из годового коэффициента.
- [ ] Не сохранять вычисленные fallback-проценты, пока пользователь не менял
  квартал вручную.
- [ ] Вывести объяснение годовых часов и источника распределения в настройках.

## 2. Безопасность и устаревшие доли

- [ ] Не принимать автоматически удалённые настройки из БД поверх отличающейся
  открытой формы; сообщать о конфликте без записи в БД.
- [ ] Убрать из формы неиспользуемые доли литья и упаковки, сохранив legacy
  значения и коэффициент потерь.

## 3. Проверка и выпуск

- [ ] Добавить smoke-проверки fallback-плана и отсутствия legacy-полей в UI.
- [ ] Прогнать проверки, поднять версию, выпустить обе копии калькулятора.
