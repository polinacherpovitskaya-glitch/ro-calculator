# Полный список заказов в подсказке загрузки производства

## Задача

В полосе «Загрузка производства» подсказка при наведении должна показывать
полный состав выбранного сегмента. Сейчас она сокращает список после восьми
строк, а сама подсказка не принимает курсор, поэтому заказ из неё нельзя
открыть.

## Поведение

- Для сегментов «Сделано», «Ещё делать» и «Некоммерческие работы» выводятся
  все строки без сокращения «ещё N…» и без обрезания названий.
- Список раскрывается вниз от полосы, может быть длиннее самой страницы и
  остаётся доступным, когда курсор переходит с полосы на список.
- Каждая строка с привязанным заказом — кнопка: по клику открывает карточку
  этого заказа. Строка «вне заказов» остаётся обычным текстом.
- Подсказка закрывается, только когда курсор ушёл и с полосы, и со списка.

## Границы

Не меняются расчёт часов, состав сегментов, сами заказы и структура
производственного бара. Это только раскрытие и навигация по уже рассчитанной
детализации.

## Проверка

Автотест подтверждает, что HTML подсказки содержит все строки без агрегата
«ещё N…» и что строки заказов имеют ссылку на карточку. Вручную проверяется
переход с полосы на список и открытие заказа.
